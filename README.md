# Animix

> A web game to train your own neuro-style monster

> Created at HackNTU 2016 hackthon

## See our demo
[Demo](https://leetcodetaiwan.github.io/animix/)

## Development
```
git clone https://github.com/LeetCodeTaiwan/animix
npm install
npm run dev
```

## Build for Production
```
npm run build
// or
sh deploy.sh
```
